------ Important Interview Discussion (VERY IMPORTANT)
Before coding, say this:

    For efficient collision detection and movement, I’ll use:
        -Deque for snake body
        -HashSet for O(1) collision lookup

    Why Deque?
        Snake operations:
            add head → front
            remove tail → back
            -Perfect deque use case.

    Why HashSet?
        Without HashSet:
            self collision = O(N)
        With HashSet: O(1)

----- Improvement 3 — Game Loop Separation
Don’t keep loop inside game logic.

Bad:
while(true) {
    move();
}

Better: keep unoccupied cells set to find from that set only

----- Improvement 4 — Thread Safety
Mention: synchronized move() or ReentrantLock
